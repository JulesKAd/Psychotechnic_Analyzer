Fenêtre d'Analyse (Niveau Actuel)

Calculez votre niveau (Stanine, Moyennes, conseils IA) uniquement sur vos dernières tentatives pour refléter votre niveau actuel (et non vos notes de débutants).

Dates Sélections (PSY 0 / PSY 1)

Indiquez les dates de vos épreuves pour afficher un compte à rebours sur le tableau de bord.

Gérer les Catégories (Tests)

Ajoutez manuellement le nom d'un test que vous n'avez pas encore passé. Il apparaîtra dans le graphique de répartition (en gris) et dans les listes.

Étalonnage des Classes (Stanines)

Pour chaque test, renseignez le score minimum (%) correspondant à chaque classe pour vous situer par rapport aux autres candidats et afficher les seuils sur vos graphiques.
